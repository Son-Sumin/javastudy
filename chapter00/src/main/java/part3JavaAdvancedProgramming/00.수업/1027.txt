#3.4 상속과 생성자
- 부모, 자식 클래스간의 생성자 실행 여부와 순서 -> 부모 > 자식
- 자식의 모든 생성자에서 부모의 특정 생성자를 명시(explicity)하지 않으면 암시적(implicity)으로 부모의 기본 생성자가 자식 생성자 코드 앞에 호츨된다.

#3.5 업캐스팅
암시적으로 가능 (부모의 것을 자식이 쓰는 것 -> overiding)
자식의 것을 부모가 사용할 때 꼭 명시하기(down casting)
